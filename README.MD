# CHE1147 Machine Learning Project
This is our repository for our machine learning project!

## Repository Map
- `data/` → dataset used for the project
- `figures/` → plots produced by running code

## Setup
To reproduce the Python environment:

```bash
conda env create -f environment.yml
conda activate eda-env
```

## Training and Analysis
The EDA, model training, and analysis can be run using the code in the Jupyter Notebook (.ipynb) file.
